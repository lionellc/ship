# 客户数据管理系统

微信小程序，简单的数据录入查询修改

## 特性
- Taro
- Typescript语法
- 小程序云函数
- 小程序云数据库

## 使用
```
git clone 
// 替换appid和数据库标识envName
cd ship/client 
npm install
// 之后就是 taro 的那一套了
```
